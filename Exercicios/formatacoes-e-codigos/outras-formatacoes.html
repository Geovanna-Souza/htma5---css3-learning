Código-fonte

Dentro do mundo da programação códigos são compartilhados a todo tempo em multiplas linguagens, por conta disso existe a tag <code> da HTML, onde delimita a região que contém código de alguma linguagem de programação. Sua principal vantagem semântica, é que essa tag indica ao navegador que tal trecho se trata de um código de computador (exibindo efeito visual tornando as letra monospace).
Porém somente essa tag não fornece a formatação do código (pois o mesmo fica extendido por toda a linha de forma contínua). Para solucionar tal situação utiliza-se a tag <pre> para tabular o código.
