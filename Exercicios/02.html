Parágrafos e quebras de linhas

Parágrafos são representados pela tag <p> (aberta) e </p> (fechada) no final.

Já as quebras de linha, são representadas pela tag <br>, que não tem fechamento.

Esse é um exemplo de
QUEBRAS
DE
LINHAS.

Agora vamos listar alguns símbolos:
&reg; para ®
&copy; para ©
&trade; pra ™

Além disso também se tem outros símbolos como o de moedas:
&euro; pra €
&pound; para £
&yen; para ¥
&cent; para ¢

Demais símbolos:
&empty; para ∅
&sum; para ∑
&Delta; para Δ
&larr; para ←
&uarr; para ↑
&rarr; para →
&darr; para ↓

É necessário lembrar que todos esses símbolos possuem ponto e vírgula no final para se tornarem válidos na hora de criar o texto.
