Terceira página

Essa é a terceira página!

Para voltar a primeira página é necessário voltar uma pasta, visto que essa página está dentro de uma pasta separada. Sendo assim no href é necessário adiconar ../+nome-arquivo para que o link seja redirecionado corretamente.

Esse tipo de comando é um de muitos presentes no linux, sendo assim aconselhavel estuda-lo em um futuro próximo.
