Aqui veremos como fazer uma aplicação de CSS3 externa

Primeiro é necessário criar um 'link:css' na porção <head> do documento, em seguida criar um documento de CSS e linkar com o 'href' do mesmo.

Dentro do documento de CSS, insere-se a regra de 'charset' com o UTF-8 (assim - @charset "UTF-8"), e dentro desse arquivo se aplica a estilização, para que tal modificação seja aplicada para diversas páginas diferentes, bastando linkar esse documento a todas as páginas.
