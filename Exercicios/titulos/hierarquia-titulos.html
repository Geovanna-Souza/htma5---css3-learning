Sobre a ordem hierárquica dos títulos...

Os títulos possuem seis níveis de hierarquia que servem para organizar o conteúdo dentro do site. E as tags <h1>, <h2>, <h3>, <h4>, <h5> e <h6> demarcam os títulos nos quais queremos dar destaque.

O <h1> (como podemos ver acima) é utilizado para tratar do assunto principal.

<h2>

É usado para títulos que são um sub-assunto do tema presente no texto que acompanha o <h1>.

<h3>

É usado para títulos que são um sub-assunto do tema presente no texto que acompanha o <h2>.

<h4>

É usado para títulos que são um sub-assunto do tema presente no texto que acompanha o <h3>.

<h5>

É usado para títulos que são um sub-assunto do tema presente no texto que acompanha o <h4>.

<h6>

É usado para títulos que são um sub-assunto do tema presente no texto que acompanha o <h5>.

Obs: dentro do editor de códigos (VS code), existe um comando que preenche os paragrágos para facilitar a visualização estrutural de um paragráfo por exemplo. Esse comando é o 'lorem', segue o exemplo abaixo.
